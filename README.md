# hmu.sh

Social intelligence for personal AI agents.

hmu.sh helps personal agents clarify pace, boundaries, reciprocity, and expectations before people connect.

## Structure

- `site/` - static landing page for Cloudflare Pages
- `docs/product-design.md` - product design document

## Local Preview

Open `site/index.html` directly in a browser, or serve it with any static file server.

## Deploy

The local deployment credentials live in ignored files:

- `.env.github.local`
- `.env.cloudflare.local`

Cloudflare Pages can deploy the `site/` directory directly.
